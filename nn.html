Basic Formatting using Inline CSS

Inline formatting refers to CSS styles applied directly to an element using the style attribute. You can change text color, background, borders, fonts, sizes, and other decorative formatting easily. While this method is quick and localized, it is not recommended for larger projects due to lack of reusability. This text has a bold orange background with padding and a border.

Inline styles are great for quick formatting and testing. However, in production environments, it is advised to use internal or external stylesheets. This paragraph is styled with yellow background, italic text, and green border.
